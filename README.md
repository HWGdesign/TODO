## PROJECT: TODO list
This is a simple web application that helps you keep track of your tasks and manage your to-do list effectively.<br>
With this application, you can easily add, update, and delete tasks as you complete them.
### Usage:
<ol>
<li>Start by adding a new task using the input field provided.
<li>Press Enter or click the "+" button to add the task to your to-do list.
<li>To mark a task as completed, click the circle checkbox next to the task.
<li>To edit a task, click the "pencil" icon and make the necessary changes.
<li>To delete a task, click the "x" icon.
<li>Your tasks are automatically saved locally, so you can revisit the application anytime and find your tasks exactly as you left them.
</ol>
